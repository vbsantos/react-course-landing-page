# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
    <th>Extra Small (xs)</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td> >= {BreakpointSizes.sm} </td>
  </tr>
  <tr>
    <th>Medium (md)</th>
    <td> >= {BreakpointSizes.md} </td>
  </tr>
  <tr>
    <th>Large (lg)</th>
    <td> >= {BreakpointSizes.lg} </td>
  </tr>
  <tr>
    <th>Extra Large (xl)</th>
    <td> >= {BreakpointSizes.xl} </td>
  </tr>
</table>

Using in styled components:

```tsx
import { breakAt, BreakpointSizes } from Breakpoints;

const Title = styled.h1`
  font-size: 1rem;

  ${breakAt(BreakpointSizes.md)} {
    front-size: 2rem;
  }
`;
```
