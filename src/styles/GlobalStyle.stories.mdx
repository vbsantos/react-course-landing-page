# GlobalStyle

CSS reset and base stylesheets definition.

```tsx
const App = () => (
  <>
    <GlobalStyle />
    <OtherComponents />
  </>
);
```

## Typography

Basic global typography

### Headings

All HTML headings, from `<h1>` to `<h6>`, are available:

<Preview>
  <Story name="Headings">
    <>
      <h1>Título 1</h1>
      <h2>Título 2</h2>
      <h3>Título 3</h3>
      <h4>Título 4</h4>
      <h5>Título 5</h5>
      <h6>Título 6</h6>
    </>
  </Story>
</Preview>

### Paragraph

Styling for common text elements:

<Preview>
  <Story name="Paragraph">

      </p>
      <p>
        <em>The quick brown fox jumps over the lazy dog.</em>
      </p>
      <p>
        <u>The quick brown fox jumps over the lazy dog.</u>
      </p>
      <p>
        <s>The quick brown fox jumps over the lazy dog.</s>
      </p>
    </>
  </Story>
</Preview>
